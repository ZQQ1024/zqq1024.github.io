e 与 φ(N) 不互素时，g = gcd(e, φ(N)，分为以下几种情况讨论：

1. 当`m^g < n`时
```
gcd(e, φ(N) = g
e = g * e1

c ≡ (m ^ (g * e1)) mod n，将 m^g 视为新的消息m1，c ≡ (m1 ^ e1) mod n

求得解密指数 e1d1 ≡ 1 mod φ(N)
```

2. e = 2，rabbin加密

3. e 与 (p-1)互素
```
ed ≡ 1 mod (p-1)(q-1)，而gcd(e, p-1) = 1

由 ed ≡ 1 mod (p-1)(q-1)定义得，ed ≡ 1 mod (p-1)，求得d

由 m ≡ c^d mod pq定义得，m ≡ c^d mod p
```
