## 图片格式

常见的图片格式文件头如下：
| 类型     | Hex                 | ISO 8859-1 |
| -------- | ------------------- | ---------- |
| GIF      | `47 49 46 38 39 61` | `GIF89a`   |
| JPG/JPEG | `FF D8 FF`          |            |
| PNG      | `89 50 4E 47`       | `‰PNG`     |
| BMP      | `42 4D`             | `BM`       |
